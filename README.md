# coding-share

一次关于字符编码的分享

## 目录使用指南

### sample-coding-files

可以对每个文件执行下面的操作：

- file
- hexdump
- cat
- vim

### notebooks

里面是几个 jupyter notebook

- coding_test_py* 是比较 Python 2 和 Python 3 下的 unicode、str、bytes 类型的
- encode_decode 是 encode 和 decode 的实验
- unicode_escape_and_json 是 unicode escape 的使用以及 unicode 在 json 中的存储方式
